Detailed Description

Virtual class for clustering on a shared nearest-neighbor graph.

This is a virtual class that takes the graph constructed by scran::BuildSNNGraph() and prepares it for use in the igraph library. Concrete subclasses should apply specific community detection algorithms to obtain a clustering on the cells.
