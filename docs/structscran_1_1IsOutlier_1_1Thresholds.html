◆ lower

std::vector<double> scran::IsOutlier::Thresholds::lower

Vector of length equal to the number of blocks. Each value contains the lower threshold used to define outliers (i.e., all values below this threshold are considered outliers).

If set_lower() was set to false, all values are set to negative infinity.

Note that these thresholds are always reported on the original scale, regardless of set_log(). Any log-transformation used during outlier identification is reversed when reporting the thresholds.
